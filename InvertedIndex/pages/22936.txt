w3-bar-item
If you want text instead of buttons inside the navigation bar, use the w3-bar-item class to get the same padding as the buttons:

Used in a horizontal bar:

  Home
  Link 1
  Link 2
  Link 3
  Text


Used in a vertical bar:

  Home
  Link 1
  Link 2
  Link 3
  Text
