How to Use Fashion Colors
Fashion Colors are easy to use. Just add a link to the right color library: 


  <link rel="stylesheet" 
  href="https://www.w3schools.com/lib/w3-colors-2025.css">


Then, add a class name to the HTML element you want to color:

Example

<div class="w3-2025-brandied-melon">Brandied Melon</div>
<div class="w3-2025-winterberry">Winterberry</div>
<div class="w3-2025-crocus">Crocus</div>
